# 概要
https://first-external-api-project.onrender.com/fib?n=5
のように、fibエンドポイントにて正の整数のクエリnを設定してアクセスすると
フィボナッチ行列( 1,1,2,3,5,... )の第n項( >=1 )の値を取得する。

# 構成
上のリンクにアクセスする(本apiに対してリクエストを送る)と、まずはnがクエリとして存在し、そのnが正の整数( 1,2,3,...,20577 )であるかどうかを確かめられたら、フィボナッチ行列の第n項を取得し、その値を含めたレスポンスをjson形式でリクエスト元に返す。

上記のように正常な入力が確認されなかったりfib以外のエンドポイントに対してリクエストを送ったりした場合は、エラーの内容を含めたメッセージをカスタマイズしてからレスポンスとしてリクエスト元に返す。
